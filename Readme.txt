Это руководство по программе CoolSport
Создатель: Артемий Морозов
Телеграмм: @tokreal

Для компиляции необходим MinGw (C++ 14 и выше).
Программа работает лишь в Windows 10.
Для компиляции достаточно запустить build.bat
Перед этим необходимо добавить в переменную середы путь до g++ (D:\MinGw\bin)
Для запуска после компиляции нужно нажимать на Sport.exe

В проекте есть сторонние библиотеки:

https://github.com/Iamnotexist/provod2
https://github.com/Iamnotexist/date

CoolSport - легкая программа, помогающая самостоятельно заниматься спортом. Она создана для того, чтобы хранить информацию об упражнениях.
Для записи информации об упражнении, необходимо выбрать дату записи (либо изменять дату стрелочками-кнопками, которые есть по обе стороны от даты в окне программы),
либо вписать нужную дату, в формате "ДД.ММ.ГГГГ", в поле для записи справа от стрелочки и нажать на кнопку Select New Date.

Для записи информации об упражнении, впишите название упражнения в длинное поле для записи, которое находится ниже стрелочек. Название упражнения не должно 
содержать цифр и круглых скобочек. Правее от поля записи названия, есть поле для записи количества выполненных упражнений. Туде следует вводить число выполненных упражнений.

Для записи, нажмите кнопку OK. Для корректировки значения количества упражнений, можно вводить отрицательные числа.

Все данные хранятся в файле data\Sport.info
Если необходимо перенести данные об упражнениях на другое устройство, достаточно лишь заменить файл data\Sport.info
